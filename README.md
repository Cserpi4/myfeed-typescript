# MyReddit (Demo)

A Reddit-style demo application built with **React** and **Redux Toolkit**.  
The app fetches data from the **Reddit JSON API** and displays posts and comments in a clean, responsive UI.

Live Demo: (coming soon)  
GitHub Repo: https://github.com/Cserpi4/myreddit-demo

---

## 🚀 Features
- Browse popular subreddits and posts
- Fetch data using the Reddit JSON API
- State management with **Redux Toolkit**
- Async data fetching with **createAsyncThunk**
- Markdown rendering for post content
- Proxy-based CORS handling
- Responsive layout

---

## 🛠️ Tech Stack
- **React**
- **Redux Toolkit**
- **JavaScript (ES6+)**
- **CSS**
- Reddit JSON API
